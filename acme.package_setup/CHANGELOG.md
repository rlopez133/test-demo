# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial release of acme.package_setup collection
- nginx_package role for package installation
- nginx_service role for service lifecycle management
- Support for Debian, Ubuntu, CentOS, and RHEL distributions
- Comprehensive documentation and examples
- Idempotent task design
- Tagging system for selective task execution
